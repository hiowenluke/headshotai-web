
## 1. 环境准备

### 1.1 虚拟环境（for dev）

创建虚拟环境：

```
python -m venv .venv
```

启动虚拟环境：

```
source ./.venv/bin/activate
```

查看虚拟环境路径：

```
python3 -c "import sys,os;print(sys.executable)"
```


### 1.2 初始化项目

```
npm install
```



## 2. 启动项目服务

./run_web
./run_auto_preview
