1. 项目规范：
- 项目背景：
    - 项目概述：本项目是 Headshot ai web app 移动版，为职业用户提供浏览并选择模特照片、上传自己的照片然后通过 AI 合成新的 Headshot。
    - 产品功能：
        - 提供 5 种 headshot 服务：
            - 1P：使用本 web app 提供的模特图，默认生成单张 headshot。
            - 20P/40P/80P：使用本 web app 提供的素材图片（场景、服装、姿势、发型），默认生成20/40/80张 headshot。
            - DIY：使用用户上传的素材照片（场景、服装、姿势、发型），默认生成单张 headshot。
            - 每种服务，用户均可以选择多张自己上传的照片，则生成的 headshot 数量相应翻倍。
            - 每种服务，有单独的金币价格，和任务耗时。

- 技术规范：
    - 技术栈：
        - 这是一个 PWA 项目，前端使用 Vue + Ionic + Vite，后端使用 Python、Flask。
        - 前端只支持手机端，不支持 pc 和平板，因此不要为兼容 pc 和平板而编写相关 css、js 代码
    - 交互规范：
        - 这是一个单页 web app，只有1个页面（主页），以及在它之上的各种弹窗（包括模拟为普通页面的全屏弹窗）。除非我特别要求，否则不要增加页面。
        - PageLikeModal.vue 是全屏弹窗，模拟普通页面，也即，它的外观看起来像普通的页面，有标题栏（标题栏可以包含文字和 tabs、图标），有退出按钮（返回按钮或者关闭按钮），内容在标题栏下面正确布局展示（例如可以有卡片列表、有分组 Items），等等。
    - 代码规范：

        - 保持目录结构、命名风格、注释密度与现有代码一致
